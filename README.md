| ---------------- | ---------------------------------------------------- |
| `head()`         | Shows the first N rows (default N=5)                 |
| `shape`          | Shows number of rows and columns                     |

1. Find all the unique **Wind Speed** values in the data.
2. Find the number of times when the **Weather is exactly Clear**.
